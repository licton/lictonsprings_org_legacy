CIS 211: Winter Quarter 2000

This site was designed and implemented through the work of students in North Seattle Community College's CIS 211 Class - part of the North Seattle's Webmaster program. For information about the CIS 211 Web Site Design class, contact Brian Bansenauer.

Students used Photo Shop to create web page comps as well as graphics for the Web. The course content required the students to use HTML to create web pages. The Web composer software Dreamweaver was also available for creating web pages. Students learned the techniques of image mapping, rollovers, cascading style sheets, tables and frames.
